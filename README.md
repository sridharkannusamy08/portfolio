# Sridhar Kannusamy Portfolio

A modern, high-performance portfolio website for Sridhar Kannusamy, built with Next.js and designed to showcase brand strategy, digital marketing expertise, services, case studies, and contact details.

## Features

- Responsive landing page for portfolio content
- Smooth scrolling and animated sections
- Service, project, testimonial, and process sections
- SEO-friendly metadata and social sharing setup
- Optimized for fast loading and mobile-first viewing

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion

## Development

```bash
npm install
npm run dev
```

Open http://localhost:3000 to view the site locally.

## Production Verification

```bash
npm run build
npm run lint
```

## Deployment

The project is ready to be deployed on Vercel with automatic GitHub redeployments enabled.
